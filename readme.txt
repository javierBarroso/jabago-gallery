== Description ==

This plugin gives you the possibility to create image galleries to display on your web page, giving you the possibility to customize the galleries in the way you like best, being able to choose between several layouts.


# Features

-	4 types of layouts
-	Responsive Design
-	Color Control
-	Content Control



# How to use it

If you are looking for an easy-to-use and feature-rich plugin to add galleries to your website, the Jabago Gallery plugin might be the best option for you. It's easy to use but packed with powerful features, allowing you to create simple to complex photo galleries. Jabago Gallery comes packed with impressive design options. Jabago Gallery is a great choice for photography websites and blogs, as well as sites that want to have robust image galleries with easy navigation. Take a look at the plugin demo below and give it a try.

== Installation ==

1. Upload the plugin folder 'jabago-gallery' to the '/wp-content/plugins/' directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress

== Frequently Asked Questions ==

= Is Jabago Gallery responsive? =

YES, Jabago Gallery will adapt to look nice on desktops, tablets, and mobile devices.

= Can I display multiple galleries on the same page?  =

YES, you can use as many galleries as you want, doesn't matter if it is on the same page or different ones.

= The plugin is missing a feature, can you add it? =

Sure! We improve Jabago Gallery continuously to fit your needs, if you have a feature request or feedback [shot us a line] abby.javi.infox5@gmail.com.
